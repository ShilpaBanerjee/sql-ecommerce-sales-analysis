# 🛒 SQL E-commerce Sales Analysis

## 📌 Overview

This project analyzes an e-commerce dataset using PostgreSQL to extract meaningful business insights such as customer behavior, sales trends, and revenue performance.

---

## 🛠 Tools & Technologies

* PostgreSQL
* SQL (Joins, Aggregations, Window Functions)
* pgAdmin

---

## 📊 Key Business Questions Solved

* What is the total revenue generated?
* Who are the top customers by spending?
* Which cities generate the highest sales?
* What are the monthly sales trends?
* Who are the repeat customers?
* Who is the top customer in each city?

---

## 🔍 SQL Concepts Used

* INNER JOIN
* GROUP BY & Aggregations
* ORDER BY
* Window Functions:

  * ROW_NUMBER()
  * DENSE_RANK()
  * LAG()

---

## 📂 Project Structure

* `queries/` → SQL scripts
* `dataset/` → Sample data
* `screenshots/` → Query outputs

---

## 📈 Sample Insights

* Identified high-value customers contributing maximum revenue
* Found repeat purchasing patterns among customers
* Observed sales growth trends over time

---

## 🚀 How to Use

1. Open PostgreSQL / pgAdmin
2. Run the SQL script
3. Execute queries to view insights

---

## 🏁 Conclusion

This project demonstrates practical SQL skills required for data analysis roles, including working with real-world scenarios and extracting actionable insights.
